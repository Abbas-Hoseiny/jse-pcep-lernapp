Funktionen

Wiederverwendbare Code-Blöcke. Deklaration, Ausdrücke, Arrow Functions, Scope, Hoisting, Rekursion.

5.1 Function Declaration

function addiere(a, b) {
  return a + b;
}

console.log(addiere(2, 3));  // 5
  • Name: addiere
  • Parameter: a, b
  • Körper: { return a + b; }
  • return: gibt Wert zurück. Ohne return → undefined

Function Declarations werden gehoistet — man kann sie schon vor der Definition aufrufen.

5.2 Function Expression

const addiere = function(a, b) {
  return a + b;
};

// Benannte Function Expression:
const fak = function fakultaet(n) {
  return n <= 1 ? 1 : n * fakultaet(n - 1);
};

Eine Function Expression ist eine Funktion als Wert — kann einer Variable zugewiesen werden. Kein Hoisting der Funktion selbst.

5.3 Arrow Functions

// Kompakte Syntax:
const addiere = (a, b) => a + b;           // implizites return
const quadrat = x => x * x;                 // ein Parameter: Klammern optional
const gruss = () => console.log("hi");     // keine Parameter: ()
const mehr = (a, b) => {                    // mit Körper:
  const s = a + b;
  return s;
};
Unterschiede zu normalen Funktionen:
  • Kein eigenes this — übernimmt es aus dem umgebenden Scope.
  • Kein arguments-Objekt.
  • Nicht als Konstruktor mit new verwendbar.

5.4 Parameter & Argumente

function greet(name = "Gast") {    // Default-Parameter
  console.log("Hallo, " + name);
}
greet();          // "Hallo, Gast"
greet("Ali");     // "Hallo, Ali"

function sum(...zahlen) {          // Rest-Parameter — sammelt in Array
  let s = 0;
  for (const z of zahlen) s += z;
  return s;
}
sum(1, 2, 3, 4);  // 10

// Spread beim Aufruf:
const arr = [1, 2, 3];
Math.max(...arr);  // 3

Wenn du weniger Argumente übergibst als Parameter, sind die fehlenden undefined:

function f(a, b) { console.log(a, b); }
f(1);  // 1 undefined

5.5 return

function doppel(x) { return x * 2; }
function ohneReturn() { console.log("hi"); }

doppel(5);        // 10
ohneReturn();     // gibt undefined zurück
ASI-Stolperfalle: 
return
  42;
// Interpretiert als: return; 42; → gibt undefined zurück!

// Besser:
return 42;

5.6 Scope (Gültigkeitsbereich)

let global = "überall";  // globaler Scope

function f() {
  let lokal = "nur in f";   // Funktions-Scope
  if (true) {
    let block = "nur im if"; // Block-Scope (let/const)
    var auchLokal = "aber in ganzer Funktion"; // var ignoriert Block!
  }
  console.log(auchLokal);   // OK
  // console.log(block);    // ReferenceError
}

Lexikalischer Scope

Eine innere Funktion kann auf Variablen der äußeren zugreifen (nicht umgekehrt):

function aussen() {
  const x = 1;
  function innen() {
    console.log(x);  // 1 — funktioniert!
  }
  innen();
}

5.7 Hoisting

// Function Declarations werden gehoistet (Name + Körper):
greet();   // funktioniert!
function greet() { console.log("hi"); }

// Function Expressions NICHT:
sag();     // TypeError: sag is not a function
var sag = function() { console.log("hi"); };

// let/const werden in TDZ gehoistet (nicht zugreifbar):
console.log(x);   // ReferenceError
let x = 5;

TDZ = Temporal Dead Zone: der Bereich zwischen Block-Beginn und Deklaration.

5.8 Rekursion

Eine Funktion, die sich selbst aufruft. Wichtig: Basisfall, sonst Endlosschleife.

function fakultaet(n) {
  if (n <= 1) return 1;          // Basisfall
  return n * fakultaet(n - 1);   // rekursiver Aufruf
}
fakultaet(5);   // 120  (5 * 4 * 3 * 2 * 1)

function fib(n) {
  if (n < 2) return n;
  return fib(n - 1) + fib(n - 2);
}
fib(10);  // 55

5.9 Callbacks

Ein Callback ist eine Funktion, die als Argument an eine andere übergeben wird.

function machWas(callback) {
  callback("Hallo");
}
machWas(function(nachricht) {
  console.log(nachricht);  // "Hallo"
});

// Mit Arrow Function:
machWas(msg => console.log(msg));

// Häufig bei Array-Methoden:
[1, 2, 3].forEach(x => console.log(x));
[1, 2, 3].map(x => x * 2);   // [2, 4, 6]

5.10 IIFE (Immediately Invoked Function Expression)

(function() {
  console.log("läuft sofort!");
})();

// Mit Arrow:
(() => console.log("auch sofort"))();

Nützlich, um einen eigenen Scope zu erzeugen.
