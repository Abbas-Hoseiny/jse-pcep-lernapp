Operatoren & Benutzerinteraktion

Arithmetik, Vergleich, Logik, Zuweisung, Precedence, prompt/alert/confirm.

3.1 Arithmetische Operatoren

5 + 2      // 7    Addition
5 - 2      // 3    Subtraktion
5 * 2      // 10   Multiplikation
5 / 2      // 2.5  Division
5 % 2      // 1    Modulo (Rest)
5 ** 2     // 25   Potenz (ES2016)

// Unäre Operatoren:
-5         // Negation
+"42"      // 42   — Umwandlung zu Number
++x        // x um 1 erhöhen (Pre-Increment)
x++        // x um 1 erhöhen (Post-Increment)
--x; x--;  // Dekrement
Pre vs Post Increment: 
let a = 5;
let b = a++;   // b = 5 (erst lesen, dann erhöhen), a = 6
let c = ++a;   // c = 7 (erst erhöhen, dann lesen), a = 7

3.2 Zuweisungsoperatoren

let x = 10;

x += 5;    // x = x + 5   → 15
x -= 3;    // x = x - 3   → 12
x *= 2;    // x = x * 2   → 24
x /= 4;    // x = x / 4   → 6
x %= 4;    // x = x % 4   → 2
x **= 3;   // x = x ** 3  → 8

// Auch logisch/nullish:
x ||= 5;   // wenn x falsy, auf 5 setzen
x &&= 5;   // wenn x truthy, auf 5 setzen
x ??= 5;   // wenn x null/undefined, auf 5 setzen

3.3 Vergleichsoperatoren

5 == "5"      // true   lose Gleichheit (mit Coercion)
5 === "5"     // false  strikte Gleichheit (ohne Coercion)
5 != "5"      // false
5 !== "5"     // true
5 > 3         // true
5 < 3         // false
5 >= 5        // true
5 <= 4        // false

// String-Vergleich (lexikographisch):
"apfel" < "banane"   // true
"B" < "a"             // true (Großbuchstaben < Kleinbuchstaben in ASCII)
Faustregel: Immer === und !== benutzen. == kann zu überraschenden Ergebnissen führen: 
0 == ""          // true
0 == "0"         // true
null == undefined // true
[] == false      // true
[] == ![]        // true !!

3.4 Logische Operatoren

true && false    // false   UND (AND)
true || false    // true    ODER (OR)
!true            // false   NICHT (NOT)

// Short-Circuit Evaluation:
// && gibt den ERSTEN falsy Wert zurück, sonst den LETZTEN
true && "hallo"      // "hallo"
0 && "hallo"          // 0
"a" && "b" && "c"    // "c"

// || gibt den ERSTEN truthy Wert zurück, sonst den LETZTEN
null || "default"     // "default"
"a" || "b"            // "a"
0 || "" || "fallback" // "fallback"

// Nullish Coalescing ?? — nur null/undefined fallen durch
null ?? "fallback"    // "fallback"
0 ?? "fallback"       // 0      (0 ist NICHT null/undefined)
"" ?? "fallback"      // ""

3.5 Ternärer Operator

Kurzschreibweise für if/else, die einen Wert zurückgibt:

// bedingung ? wennTrue : wennFalse

let alter = 20;
let status = alter >= 18 ? "erwachsen" : "minderjährig";

// Verschachtelbar (aber vorsichtig):
let note = punkte >= 90 ? "A"
         : punkte >= 80 ? "B"
         : punkte >= 70 ? "C" : "F";

3.6 Bitweise Operatoren (Überblick)

5 & 3    // 1   AND:   0101 & 0011 = 0001
5 | 3    // 7   OR:    0101 | 0011 = 0111
5 ^ 3    // 6   XOR:   0101 ^ 0011 = 0110
~5       // -6  NOT:   ~0101 = ...1010
5 << 1   // 10  Left-Shift  (× 2)
5 >> 1   // 2   Right-Shift (÷ 2, signed)
5 >>> 1  // 2   Unsigned Right-Shift

In der Praxis selten gebraucht, aber die Prüfung kann sie abfragen.

3.7 Operator Precedence (Vorrang)

Operatoren haben eine Reihenfolge. Wichtige (hoch → niedrig):

  1. () Klammern
  2. ++, --, !, typeof, unäres -
  3. ** Potenz
  4. *, /, %
  5. +, -
  6. <, <=, >, >=
  7. ==, !=, ===, !==
  8. &&
  9. ||, ??
  10. ? : ternär
  11. =, +=, … (Zuweisung)
Tipp: Im Zweifel Klammern setzen — bessere Lesbarkeit und keine Überraschungen.

3.8 Benutzerinteraktion (nur im Browser!)

alert("Hallo!");
// Popup mit OK-Button. Rückgabe: undefined.

const name = prompt("Wie heißt du?", "Standard");
// Popup mit Eingabefeld. Rückgabe: String oder null (bei Abbruch).

const ok = confirm("Bist du sicher?");
// Popup mit OK/Abbrechen. Rückgabe: true oder false.
Wichtig für die Prüfung:
  • prompt() gibt IMMER einen String zurück — ggf. mit Number(...) umwandeln!
  • Wird „Abbrechen" gedrückt, gibt prompt null zurück.
  • Diese Funktionen gibt es nur im Browser (nicht in Node.js).
const alterStr = prompt("Wie alt bist du?");
const alter = Number(alterStr);
if (alter >= 18) alert("Erwachsen!");

3.9 Der typeof- und delete-Operator

typeof 42;         // "number"   — gibt den Typ als String zurück

let obj = { a: 1, b: 2 };
delete obj.a;      // entfernt die Eigenschaft a
console.log(obj);  // { b: 2 }
