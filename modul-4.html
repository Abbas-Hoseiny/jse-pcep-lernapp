Kontrollfluss: if, switch, Schleifen

Bedingungen und Schleifen sind das Rückgrat jedes Programms.

4.1 if / else if / else

if (bedingung) {
  // führt aus wenn bedingung truthy
}

if (alter >= 18) {
  console.log("erwachsen");
} else if (alter >= 13) {
  console.log("teenager");
} else {
  console.log("kind");
}

Die Bedingung wird in einen Boolean umgewandelt (Truthiness-Regeln gelten).

Ohne Block-Klammern führt if nur die nächste Anweisung aus: 
if (x) console.log("a");
  console.log("b");  // immer ausgeführt!
→ Lieber immer { } verwenden.

4.2 switch / case

switch (wert) {
  case 1:
    console.log("eins");
    break;              // ohne break: "Fall-through"!
  case 2:
  case 3:
    console.log("zwei oder drei");
    break;
  default:
    console.log("etwas anderes");
}
Wichtig:
  • switch nutzt strikte Gleichheit (===).
  • Ohne break läuft der Code in den nächsten case hinein (Fall-through).
  • default ist optional und kann überall stehen (üblich: am Ende).

4.3 while-Schleife

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
// Ausgabe: 0 1 2 3 4

Prüft die Bedingung vor jedem Durchlauf. Kann 0× ausgeführt werden.

Endlosschleife! Wenn du i++ vergisst, läuft die Schleife unendlich. Der Browser friert ein.

4.4 do...while-Schleife

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Prüft die Bedingung nach jedem Durchlauf. Läuft mindestens einmal.

4.5 for-Schleife

for (init; bedingung; update) {
  // Körper
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}
// 0 1 2 3 4

// Alle drei Teile sind optional:
for (;;) { /* Endlosschleife */ }

Reihenfolge pro Iteration:

  1. Init (nur beim ersten Mal)
  2. Bedingung prüfen — wenn false, Schleife verlassen
  3. Körper ausführen
  4. Update
  5. Zurück zu Schritt 2

4.6 for...of und for...in

const arr = ["a", "b", "c"];

// for...of — über WERTE iterieren (Arrays, Strings, ...)
for (const wert of arr) {
  console.log(wert);   // "a", "b", "c"
}

// for...in — über SCHLÜSSEL/INDIZES iterieren (meist Objekte)
for (const key in { a: 1, b: 2 }) {
  console.log(key);    // "a", "b"
}
Faustregel: Für Arrays bevorzuge for...of oder forEach. for...in bei Objekten.

4.7 break und continue

// break — Schleife komplett beenden
for (let i = 0; i < 10; i++) {
  if (i === 3) break;
  console.log(i);   // 0 1 2
}

// continue — nur aktuelle Iteration überspringen
for (let i = 0; i < 5; i++) {
  if (i % 2 === 0) continue;
  console.log(i);   // 1 3
}

4.8 Verschachtelte Schleifen

for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(i, j);
  }
}
// 1 1, 1 2, 1 3, 2 1, 2 2, 2 3, 3 1, 3 2, 3 3

break verlässt nur die innerste Schleife. Mit Labels kann man mehrere verlassen:

outer: for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) break outer;
    console.log(i, j);
  }
}
