Tipps zur Nutzung

  • Fortschritt wird im Browser gespeichert (localStorage) — keine Anmeldung nötig.
  • Lernkarteien: Leertaste = umdrehen, ←/→ = navigieren. „Wusste ich" entfernt die Karte aus dem Stapel.
  • Code-Playground: Cmd/Ctrl+Enter führt Code aus. Nutze console.log() für Ausgaben.
  • Quiz-Fragen haben Erklärungen — lies sie auch wenn du richtig geantwortet hast.
  • Prüfungssimulation mehrfach machen, bis du konstant > 80 % erreichst.
