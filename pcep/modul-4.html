Kontrollfluss: Bedingungen & Schleifen

if/elif/else, while, for, range(), break, continue, while-else, for-else, pass.

4.1 if / elif / else

if bedingung:
    # Code
elif andere_bedingung:
    # Code
else:
    # Code

Beachte: Doppelpunkt : am Ende, dann Einrückung für den Block.

alter = 25
if alter >= 18:
    print("erwachsen")
elif alter >= 13:
    print("teenager")
else:
    print("kind")

elif = „else if”. Beliebig viele erlaubt. else ist optional, kann nur einmal am Ende stehen.

4.2 Verschachtelte Bedingungen

if alter >= 18:
    if hat_ausweis:
        print("darf rein")
    else:
        print("Ausweis vergessen")
else:
    print("zu jung")

Jede Verschachtelung = eine weitere Einrückungsebene.

4.3 while-Schleife

i = 0
while i < 5:
    print(i)
    i += 1
# 0 1 2 3 4

Prüft die Bedingung vor jedem Durchlauf. Kann 0× laufen.

Endlosschleife! Wenn du i += 1 vergisst, läuft sie unendlich.

4.4 for-Schleife — iteriert über Sequenzen

for zeichen in "Hallo":
    print(zeichen)
# H a l l o

for zahl in [1, 2, 3]:
    print(zahl)

for key in {"a": 1, "b": 2}:    # iteriert über Keys
    print(key)
Großer Unterschied zu JS: Python hat keine C-Style-for-Schleife. Immer über Sequenzen!

4.5 range()

Erzeugt eine Zahlenfolge — perfekt für for.

range(5)         # 0, 1, 2, 3, 4          (5 Werte)
range(1, 6)      # 1, 2, 3, 4, 5          (start, stop)
range(0, 10, 2)  # 0, 2, 4, 6, 8          (start, stop, step)
range(10, 0, -1) # 10, 9, 8, ..., 1

for i in range(5):
    print(i)
# 0 1 2 3 4
range()-Regeln:
  • stop ist ausschließlich — nicht enthalten!
  • range erzeugt ein iterierbares Objekt, keine Liste. Umwandeln: list(range(5)).
  • Rückwärts nur mit negativem step.

4.6 break und continue

# break — Schleife komplett verlassen
for i in range(10):
    if i == 5:
        break
    print(i)
# 0 1 2 3 4

# continue — nur aktuelle Iteration überspringen
for i in range(5):
    if i == 2:
        continue
    print(i)
# 0 1 3 4

4.7 while-else und for-else — Python-Spezialität!

Der else-Block wird ausgeführt, wenn die Schleife normal endete (nicht durch break).

for i in range(5):
    print(i)
else:
    print("Schleife komplett durchgelaufen")
# 0 1 2 3 4
# Schleife komplett durchgelaufen

# Mit break — else läuft NICHT:
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("nicht erreicht")
# 0 1 2
PCEP-Klassiker: nützlich für „Suche”-Schleifen. for-Schleife sucht etwas — wenn nicht gefunden (kein break), wird der else-Block ausgeführt.

4.8 pass — die „leere” Anweisung

Python braucht bei leeren Blöcken einen Platzhalter — Einrückung allein reicht nicht.

if x > 0:
    pass           # TODO: später implementieren

def spaeter():
    pass           # leere Funktion

for i in range(10):
    pass           # leere Schleife

4.9 Verschachtelte Schleifen

for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
# 1 1, 1 2, 1 3, 2 1, 2 2, 2 3, 3 1, 3 2, 3 3

# break verlässt nur die INNERSTE Schleife
for i in range(3):
    for j in range(3):
        if j == 1:
            break
        print(i, j)
# 0 0, 1 0, 2 0

Python hat keine Labels für break. Um aus mehreren Schleifen auszubrechen: Flag, Funktion mit return, oder Exception.
