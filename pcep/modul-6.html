6.1 Funktionen definieren

def addiere(a, b):
    return a + b

print(addiere(2, 3))   # 5
  • def = definiere Funktion
  • name(params): — Name + Klammern + Parameter + Doppelpunkt
  • Eingerückter Körper
  • return gibt einen Wert zurück (optional)

Ohne return gibt die Funktion implizit None zurück.

6.2 Docstrings

def quadrat(x):
    """Gibt das Quadrat von x zurück."""
    return x * x

help(quadrat)
print(quadrat.__doc__)   # "Gibt das Quadrat von x zurück."

Ein Docstring ist der erste String in einer Funktion — wird für Doku verwendet.

6.3 Parameter und Argumente

  • Parameter: die Namen in der def-Signatur.
  • Argumente: die Werte, die beim Aufruf übergeben werden.

Positionale Argumente

def info(name, alter):
    print(f"{name} ist {alter}")

info("Ali", 25)   # nach Position

Keyword-Argumente

info(alter=25, name="Ali")   # nach Name — Reihenfolge egal

Default-Parameter

def gruss(name, gruss_text="Hallo"):
    print(f"{gruss_text}, {name}!")

gruss("Ali")                      # Hallo, Ali!
gruss("Ali", "Hi")                # Hi, Ali!
gruss("Ali", gruss_text="Hey")    # Hey, Ali!
FALLE: Default-Wert wird nur einmal erzeugt. Nie mutable Defaults (= []) verwenden! 
def bad(l=[]):
    l.append(1)
    return l
bad()   # [1]
bad()   # [1, 1] !!!

6.4 *args und **kwargs

def summe(*args):         # args ist ein TUPEL
    s = 0
    for x in args:
        s += x
    return s
summe(1, 2, 3, 4)          # 10

def info(**kwargs):        # kwargs ist ein DICT
    for k, v in kwargs.items():
        print(f"{k} = {v}")
info(name="Ali", alter=25)

def alles(a, b, *args, **kwargs):
    print(a, b, args, kwargs)
alles(1, 2, 3, 4, x=10, y=20)
# 1 2 (3, 4) {'x': 10, 'y': 20}

6.5 Unpacking beim Aufruf

def add(a, b, c):
    return a + b + c

l = [1, 2, 3]
d = {"a": 1, "b": 2, "c": 3}

add(*l)       # = add(1, 2, 3)
add(**d)      # = add(a=1, b=2, c=3)

6.6 Scope und die LEGB-Regel

Python sucht Namen in dieser Reihenfolge:

  1. Local — innerhalb der aktuellen Funktion
  2. Enclosing — umgebende Funktion (bei Nesting)
  3. Global — Modul-Ebene
  4. Built-in — Python selbst (print, len, …)
x = "global"

def aussen():
    x = "enclosing"
    def innen():
        x = "local"
        print(x)     # local
    innen()
    print(x)         # enclosing

aussen()
print(x)             # global

6.7 global und nonlocal

zaehler = 0

def erhoehe():
    global zaehler       # verweist auf globales zaehler
    zaehler += 1

erhoehe()
print(zaehler)   # 1

def aussen():
    x = 10
    def innen():
        nonlocal x        # verweist auf enclosing x
        x = 99
    innen()
    print(x)              # 99
aussen()
Ohne global: Eine Zuweisung in der Funktion erzeugt eine lokale Variable — die globale bleibt unverändert.

6.8 Rekursion

def fakultaet(n):
    if n <= 1:           # Basisfall!
        return 1
    return n * fakultaet(n - 1)

print(fakultaet(5))   # 120

def fib(n):
    if n < 2:
        return n
    return fib(n-1) + fib(n-2)

print(fib(10))   # 55
RecursionError: Python hat ein Rekursionslimit (standardmäßig 1000). Zu tief → Fehler.

6.9 Funktionen sind erste-Klasse-Objekte

def doppel(x): return x * 2

f = doppel                  # Zuweisung
f(5)                         # 10

def wende_an(fn, x):        # Funktion als Argument
    return fn(x)
wende_an(doppel, 7)         # 14

# Lambda — anonyme Funktion
quadrat = lambda x: x * x
quadrat(4)                   # 16

list(map(lambda x: x*2, [1,2,3]))     # [2,4,6]
list(filter(lambda x: x>2, [1,2,3,4])) # [3,4]

6.10 Mehrere Rückgabewerte (per Tupel)

def minmax(liste):
    return min(liste), max(liste)   # Tuple

lo, hi = minmax([3, 1, 4, 1, 5])
print(lo, hi)   # 1 5
