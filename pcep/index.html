Unterschiede zu JavaScript

  • Indentation statt Klammern: Python nutzt Einrückung (4 Leerzeichen) für Blöcke — nicht { }.
  • Keine Semikolons am Zeilenende.
  • Strenge Typisierung: "5" + 3 wirft TypeError — nicht "53".
  • Nur == — kein ===, Python vergleicht immer strikt.
  • None statt null, True/False statt true/false (Großbuchstaben!).
  • and/or/not statt &&/||/!.
