Variablen & Datentypen

Datentypen (int, float, bool, str, None), Typumwandlung, input().

2.1 Variablen deklarieren

In Python gibt es kein var, let oder const. Du weist einfach zu:

name = "Ali"
alter = 25
groesse = 1.75
  • Typ wird automatisch aus dem Wert abgeleitet.
  • Variablen existieren ab der ersten Zuweisung.
  • Neu-Zuweisung darf den Typ wechseln (dynamic typing).
x = 5        # int
x = "hi"     # jetzt str — völlig ok
Python kennt keine echten Konstanten. Konvention: UPPER_CASE zeigt „nicht verändern” an, wird aber nicht erzwungen.

2.3 int — Ganzzahl

a = 42
b = -17
groß = 10**100        # Python hat keine int-Überlaufgrenze!

# Unterstriche zur Lesbarkeit:
zahl = 1_000_000      # = 1000000

int in Python ist beliebig groß — keine 32/64-bit-Grenze.

2.4 float — Gleitkomma

pi = 3.14159
e = 1.5e3             # 1500.0 (wissenschaftlich)
klein = 1.5e-3        # 0.0015
unendlich = float("inf")
nix = float("nan")

# Gleitkomma-Ungenauigkeit!
0.1 + 0.2 == 0.3      # False !
0.1 + 0.2             # 0.30000000000000004
Achtung: Wie in allen IEEE-754-Sprachen. Für exakte Dezimalzahlen: decimal.Decimal.

2.5 bool — Boolean

ist_ok = True              # Großbuchstabe!
fertig = False

# bool ist ein Subtyp von int:
True == 1                  # True
False == 0                 # True
True + True                # 2
isinstance(True, int)      # True

Truthy / Falsy

Falsy (werden zu False): None, False, 0, 0.0, "", [], (), {}, set().

Alles andere ist truthy. Leere Container sind in Python falsy — anders als in JavaScript!

bool([])     # False
bool([0])    # True  (nicht leer!)
bool("")     # False
bool(" ")    # True

2.6 str — Strings

a = "doppelt"
b = 'einfach'
c = """dreifach
  mehrzeilig"""
d = '''auch mehrzeilig'''

# f-String (ab Python 3.6):
name = "Ali"
print(f"Hallo, {name}!")      # "Hallo, Ali!"
print(f"Summe: {2 + 3}")       # "Summe: 5"

# Alte Formatierungen:
print("Hallo, {}!".format(name))
print("Hallo, %s!" % name)

# Escape-Sequenzen:
"Zeile1\nZeile2"    # Zeilenumbruch
"Tab\there"          # Tab
"Er \"sagte\"."     # doppeltes Anführungszeichen
"Backslash: \\"     # Backslash
r"C:\Users"         # raw string — \ nicht escaped
Strings sind immutable: s[0] = "X" wirft TypeError.

Wichtige String-Methoden

"hallo".upper()         # 'HALLO'
"HALLO".lower()          # 'hallo'
"  hi  ".strip()         # 'hi'
"hallo".replace("l", "L") # 'haLLo'
"a,b,c".split(",")       # ['a', 'b', 'c']
",".join(["a","b","c"]) # 'a,b,c'
"hallo".startswith("ha") # True
"hallo".endswith("lo")   # True
"hallo".find("ll")       # 2   (-1 wenn nicht gefunden)
"hallo".count("l")       # 2
len("hallo")             # 5

2.7 None

x = None
print(x)          # None
type(x)           # <class 'NoneType'>

# Prüfung IMMER mit 'is', nicht '==':
if x is None:
    print("leer")

None repräsentiert „kein Wert”. Funktionen ohne return geben implizit None zurück.

2.8 type() — Typ einer Variable prüfen

type(42)           # <class 'int'>
type(3.14)         # <class 'float'>
type("hi")         # <class 'str'>
type(True)         # <class 'bool'>
type(None)         # <class 'NoneType'>
type([])           # <class 'list'>

# Prüfung: isinstance (empfohlen)
isinstance(42, int)             # True
isinstance(True, int)           # True !  (bool ist Subtyp)
isinstance(42, (int, float))    # True — mehrere Typen

2.9 Typumwandlung (Casting)

int("42")         # 42
int(3.9)          # 3   — abschneiden, NICHT runden
int(True)         # 1
int("0xff", 16)   # 255  — mit Basis
int("abc")        # ValueError!

float("3.14")     # 3.14
float(5)          # 5.0
float("inf")      # inf

str(42)           # '42'
str(3.14)         # '3.14'
str(True)         # 'True'
str([1,2])        # '[1, 2]'

bool(0)           # False
bool(1)           # True
bool("")          # False
bool("False")     # True !  (nicht leer)
bool([])          # False
bool(None)        # False
Keine implizite Konvertierung wie in JS! "5" + 3 wirft TypeError. Python zwingt dich, explizit umzuwandeln.

2.10 input() — Benutzereingabe

name = input("Wie heißt du? ")
print(f"Hallo, {name}!")
Wichtig: input() gibt IMMER einen String zurück! 
alter = input("Alter: ")         # str!
alter_zahl = int(alter)           # explizit umwandeln
# oder kombiniert:
alter = int(input("Alter: "))
