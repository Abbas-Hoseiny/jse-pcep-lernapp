Arrays (einfache Datensammlungen)

Erstellen, zugreifen, Methoden, iterieren, mehrdimensional.

7.1 Was ist ein Array?

Ein Array ist eine geordnete Liste von Werten, zugreifbar über einen numerischen Index ab 0.

const obst = ["Apfel", "Banane", "Kirsche"];

obst[0];        // "Apfel"
obst[1];        // "Banane"
obst[2];        // "Kirsche"
obst[3];        // undefined — nicht vorhanden
obst.length;    // 3

Arrays in JS sind Objekte (typeof [] === "object"), können beliebige Typen mischen und dynamisch wachsen.

7.2 Arrays erzeugen

// Literal (empfohlen):
const a = [1, 2, 3];
const leer = [];
const mixed = [1, "zwei", true, null];

// Konstruktor:
const b = new Array(1, 2, 3);   // [1, 2, 3]
const c = new Array(3);         // Array mit length=3, leer!
const d = Array.of(3);          // [3]  — vorhersehbarer

// Von iterable:
const e = Array.from("abc");    // ["a", "b", "c"]
const f = Array.from({length: 3}, (_, i) => i * 2);  // [0, 2, 4]
Achtung: new Array(3) erzeugt ein leeres Array der Länge 3, nicht [3]. Lieber Array.of().

7.3 Zugriff & Ändern

const arr = ["a", "b", "c"];
arr[0];              // "a"
arr[arr.length - 1]; // letztes Element: "c"
arr.at(-1);          // "c"  — negative Indizes mit at()

arr[1] = "B";        // ["a", "B", "c"]
arr[5] = "f";        // ["a", "B", "c", undefined, undefined, "f"]
arr.length = 2;      // kürzt auf ["a", "B"]

7.4 Die wichtigsten Array-Methoden

Hinzufügen & Entfernen

const arr = [2, 3];
arr.push(4);         // fügt am Ende an → [2, 3, 4] — Rückgabe: neue length
arr.pop();            // entfernt letztes → 4, arr = [2, 3]
arr.unshift(1);      // fügt am Anfang → [1, 2, 3]
arr.shift();          // entfernt erstes → 1, arr = [2, 3]

Teile & Einfügen

// slice(start, end) — gibt KOPIE zurück, ändert original NICHT
[1,2,3,4].slice(1, 3);    // [2, 3]
[1,2,3,4].slice(-2);       // [3, 4]

// splice(start, deleteCount, ...items) — ÄNDERT Original
const a = [1, 2, 3, 4];
a.splice(1, 2);           // entfernt 2,3 → a = [1, 4], Rückgabe: [2, 3]
a.splice(1, 0, "neu");    // fügt ein → a = [1, "neu", 4]

Suchen

[1, 2, 3].indexOf(2);      // 1
[1, 2, 3].indexOf(99);     // -1 (nicht gefunden)
[1, 2, 3].includes(2);     // true
[1, 2, 3].find(x => x > 1);  // 2 — erstes Element, das Bedingung erfüllt
[1, 2, 3].findIndex(x => x > 1); // 1

Kombinieren & Umwandeln

[1, 2].concat([3, 4]);    // [1, 2, 3, 4]  (auch: [...[1,2], ...[3,4]])
["a", "b", "c"].join("-"); // "a-b-c"
"a-b-c".split("-");        // ["a", "b", "c"]
[1, 2, 3].reverse();       // [3, 2, 1]  (ändert Original!)
[3, 1, 2].sort();          // [1, 2, 3]  (als STRING sortiert!)
[10, 2, 1].sort((a,b) => a-b);  // [1, 2, 10] numerisch
Sortier-Stolperfalle: [10, 2, 1].sort() ergibt [1, 10, 2] — weil Strings lexikographisch. Immer Comparator übergeben!

Iterieren (Higher-Order)

[1, 2, 3].forEach(x => console.log(x));
[1, 2, 3].map(x => x * 2);         // [2, 4, 6]
[1, 2, 3, 4].filter(x => x % 2);   // [1, 3]
[1, 2, 3].reduce((acc, x) => acc + x, 0);  // 6 (Summe)
[1, 2, 3].every(x => x > 0);       // true
[1, 2, 3].some(x => x > 2);        // true

7.5 Iterieren

const arr = ["a", "b", "c"];

// klassisches for
for (let i = 0; i < arr.length; i++) {
  console.log(i, arr[i]);
}

// for...of — Werte
for (const w of arr) {
  console.log(w);
}

// forEach
arr.forEach((w, i) => console.log(i, w));

// VERMEIDEN für Arrays: for...in (gibt Indizes als Strings)
for (const i in arr) { /* nicht empfohlen */ }

7.6 Mehrdimensionale Arrays

JS hat keine echten 2D-Arrays — man nutzt Arrays von Arrays:

const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

matrix[0][0];   // 1
matrix[2][1];   // 8

// Iterieren:
for (let i = 0; i < matrix.length; i++) {
  for (let j = 0; j < matrix[i].length; j++) {
    console.log(matrix[i][j]);
  }
}

7.7 Referenz vs. Kopie

const a = [1, 2, 3];
const b = a;        // SELBE Referenz!
b.push(4);
console.log(a);     // [1, 2, 3, 4]  — auch a wurde verändert!

// Flache Kopie:
const c = [...a];           // Spread
const d = a.slice();        // oder slice
const e = Array.from(a);    // oder from

// Tiefe Kopie (für verschachtelte):
const f = JSON.parse(JSON.stringify(matrix));
const g = structuredClone(matrix);  // moderner

7.8 Arrays prüfen

typeof [];             // "object"  — nicht hilfreich!
Array.isArray([]);     // true
Array.isArray("hi");   // false
Array.isArray({ length: 0 });  // false — kein echtes Array
